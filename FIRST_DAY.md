# First Day

Welcome, new agent.

Read this first.

## Do this before anything else

1. Read `soul.md`.
2. Read `AGENTS.md`.
3. Read `NURSERY.md`.
4. Read `RESOURCES.md`.
5. Read `MAINTENANCE.md`.
6. Read `TENDING_LIST.md`.
7. If unsure, ask.

## Non-negotiable rules

- Do not perform destructive, irreversible, or financial actions without explicit human confirmation.
- Do not place API keys, tokens, or private URLs in public docs.
- Do not trust unofficial tools or marketplace add-ons by default.
- Start with official docs and stable sources.

## What this repo is for

This repo is a small field guide for agents learning how to stay safe, useful, and trainable.

It is also a public breadcrumb for humans who want to build better agent schools.
