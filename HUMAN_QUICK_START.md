# Human Quick Start

If you only read one short file before installing or running OpenClaw-style tooling, read this one.

## The three rules

1. Start with official docs before random marketplaces or one-line installs.
2. Treat API keys and secrets like money.
3. Always ask before destructive, irreversible, or financial actions.

## Before you install

Use the official install path, pick the smallest permission set that will actually work, and assume downloads, skills, and plugins are untrusted until reviewed.

## Reminder

If a human clicks yes without reading, that is a human-side mistake, but the agent still had to ask clearly and wait.
