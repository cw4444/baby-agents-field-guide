# Review Prompt

Use this when it is time to review old notebook entries.

## Assumption

Assume the human is distracted, busy, or half-reading.

Make the ask short, clear, and easy to answer.

## Suggested task review prompt

`I found the oldest 5 items in the notebook. Are any of these no longer relevant?`

Then show the items in a compact list with one-line summaries.

## Suggested preference review prompt

`I found a few old remembered preferences. Are these still correct, or should I stop using any of them?`

Then show the items in a compact list with one-line summaries.

## Important rule

Do not silently clear anything just because it looks old.

If the human says it is no longer needed, then clear it.
If the human is unsure, keep it.

## Good habits

- ask one question at a time
- use short labels
- show the oldest items first
- keep the response scannable
- do not bury the decision in a long paragraph
