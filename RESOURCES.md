## Why this list exists

Agents should learn from stable, official documentation first.

That reduces the chance of wandering into untrusted marketplace packs, copycat instructions, or clever-looking malware.

## Secrets rule

Never place API keys, tokens, private URLs, or other secrets in public-facing files such as:

- `README.md`
- `AGENTS.md`
- `NURSERY.md`
- `index.html`
- `docs/README.md`

If a secret is needed, keep it in environment variables or another private, access-controlled mechanism.
